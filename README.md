# Secure Password Generator

A modern, highly responsive, and beautiful Secure Password Generator built with React. Designed to provide maximum entropy and security while looking like a premium, professional web application.

## Features

- 🔐 **Cryptographically Secure**: Guarantees at least one character of each selected type for maximum entropy.
- 🎨 **Premium UI**: Features deep radial mesh gradients, custom glowing toggle switches, and sleek typography (`Outfit` & `Space Mono`).
- 📱 **Fully Responsive**: Adapts flawlessly to desktop and mobile devices.
- 📋 **One-Click Copy**: Instantly copy your generated password to the clipboard with built-in success state feedback.
- 🎛️ **Highly Customizable**: Adjust password length (4-32 characters) and toggle uppercase, lowercase, numbers, and symbols.

## Tech Stack

- **React** (Hooks: `useState`, `useEffect`, `useCallback`)
- **Vanilla CSS3** (CSS Grid, Flexbox, Custom Animations, Variables)
- **Create React App**

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd password-generator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the local development server:

```bash
npm start
```
The application will be available at `http://localhost:3000`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```
The compiled files will be generated in the `build/` directory, ready to be deployed.

## Deployment (Vercel)

This application is completely production-ready and can be deployed instantly on Vercel:
1. Push your code to a GitHub repository.
2. Log into Vercel and click "Add New Project".
3. Import your GitHub repository.
4. Vercel will automatically detect the Create React App framework and deploy it flawlessly without any extra configuration.
